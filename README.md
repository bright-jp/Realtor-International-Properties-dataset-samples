この **Realtor International Properties Dataset Sample** には **1,000件のレコード** が含まれており、**Bright Data API** を使用して抽出されています。

## Data Points

| Field Name | Description |
|------------|-------------|
| `url` | Realtor上の物件詳細掲載ページへの直接URLです。 |
| `id` | 各物件掲載に割り当てられる一意の識別子です。 |
| `title` | 住所の詳細を含む、物件掲載のタイトルまたは見出しです。 |
| `price` | 指定された通貨における物件の掲載価格です。 |
| `currency` | 物件価格の通貨単位です。 |
| `description` | 物件の特徴や設備に関する簡単な説明または概要です。 |
| `images` | 物件を紹介する画像のURLで、リストとして提供されます。 |
| `bedrooms` | 物件の寝室数です。 |
| `bathrooms` | 物件のバスルーム数です。 |
| `property_type` | 物件タイプ（例：house、apartment、villa）です。 |
| `building_size` | 建物の総面積（平方フィートまたはメートル）です。 |
| `land_size` | 物件が建てられている土地の総面積です。 |
| `rooms` | 物件の総部屋数です。 |
| `address` | 物件の完全な住所です。 |
| `country_code` | 物件所在地の国を表すISOコードです。 |
| `state` | 物件が所在する州または地域です。 |
| `city` | 物件が所在する都市です。 |
| `realtor_details` | 物件掲載を管理する不動産業者または代理店に関する情報です。 |
| `price_history` | 物件価格の時間経過に伴う変更履歴データです。 |
| `listing_status` | 掲載の現在ステータス（例：active、pending、sold）です。 |


## Use Cases

### 1. Market Analysis
このデータセットを活用して包括的な市場分析を行い、さまざまな地域におけるトレンドや価格戦略を特定できます。

### 2. Investment Opportunities
物件詳細と過去の価格データを分析することで、有望な不動産投資機会を特定できます。

### 3. Competitive Benchmarking
データセットを使用して物件掲載を比較し、不動産市場における競争上のポジショニングを評価できます。
